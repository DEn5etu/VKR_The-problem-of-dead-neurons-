# VKR_The-problem-of-dead-neurons-
Репозиторий содержит код программы, которая выполняет различные вычислительные исследования стандартной и модифицированной моделей Кротова-Хопфилда

В рамках практической части работы был разработан вычислительный прототип для моделирования исходной и модифицированной динамики современной сети Хопфилда. Программа предназначена для численной проверки эффектов, связанных с мёртвыми нейронами, плоскими направлениями функционала энергии и применением SLR-критерия в активном подпространстве.

Программа выполняет следующие основные действия.

\begin{enumerate}
    \item Создаёт случайные двумерные модели для функций активации ReLU и softmax. Для каждой модели случайно формируются матрица связей \(W\) и вектор смещения \(b\). Матрица \(W\) масштабируется так, чтобы её спектральная норма была примерно равна \(0.65\).

    \item Реализует исходную динамику Кротова--Хопфилда:
    \[
        \dot y = Wg(y)-y+b.
    \]

    \item Реализует исходный энергетический функционал:
    \[
        E(y)
        =
        (y-b)^\top g(y)
        -
        L(y)
        -
        \frac12 g(y)^\top Wg(y).
    \]

    \item Реализует модифицированную динамику:
    \[
        \dot y
        =
        (I-W\Lambda(y))(Wg(y)-y+b),
    \]
    где
    \[
        \Lambda(y)=Dg(y)=\nabla^2L(y).
    \]

    \item Реализует модифицированную энергию:
    \[
        E_3(y)
        =
        \frac12\|Wg(y)-y+b\|^2.
    \]

    \item Строит векторные поля и линии уровня энергии для ReLU и softmax. Для каждой функции активации сравниваются стандартная и модифицированная энергии.

    \item Проверяет наличие плоских направлений энергии. Для ReLU проверяется направление мёртвого нейрона, а для softmax --- направление \((1,1)\), соответствующее сдвиговой инвариантности.

    \item Вычисляет проекторы на активное и плоское подпространства:
    \[
        P_a:\Range\Lambda(y),
        \qquad
        P_d:\Ker\Lambda(y).
    \]

    \item Проверяет SLR-критерий Смита на сетке точек:
    \[
        \lambda_1(J_{a,s})+\lambda_2(J_{a,s})<0.
    \]
    Дополнительно программа подсчитывает количество точек сетки, в которых это условие выполнено, и долю таких точек среди всех точек сетки.

    \item Находит состояния равновесия для текущей случайной матрицы \(W\). Эти равновесия отмечаются на графиках траекторий.

    \item Сравнивает исходную и модифицированную динамику по траекториям, конечной невязке и доле успешных сходимостей.
\end{enumerate}

Программа запускается из корня проекта командой:
\begin{verbatim}
python -m src.py_experiments --out outputs --grid 61 --n-ic 80
\end{verbatim}

Здесь параметр
\begin{verbatim}
--out outputs
\end{verbatim}
задаёт папку, в которую сохраняются все графики, таблицы и служебные файлы.

Параметр
\begin{verbatim}
--grid 61
\end{verbatim}
задаёт размер сетки для проверки SLR-критерия. Например, значение \(61\) означает, что проверка проводится на сетке \(61\times 61\).

Параметр
\begin{verbatim}
--n-ic 80
\end{verbatim}
задаёт число случайных начальных условий, используемых для сравнения исходной и модифицированной динамики.

Если необходимо зафиксировать случайную матрицу \(W\), чтобы результаты были воспроизводимыми, используется параметр \texttt{--w-seed}. Например:
\begin{verbatim}
python -m src.py_experiments --out outputs --grid 61 --n-ic 80 --w-seed 123
\end{verbatim}

После запуска в папке \texttt{outputs} формируются основные файлы результатов:
\begin{verbatim}
figure1_replica.png
flatness_relu.png
flatness_softmax.png
slr_heatmap.png
smith_condition_summary.csv
smith_condition_fraction.png
trajectories_original_relu.png
trajectories_modified_relu.png
trajectories_original_softmax.png
trajectories_modified_softmax.png
dynamics_comparison.png
equilibria_relu.csv
equilibria_softmax.csv
model_parameters_relu.json
model_parameters_softmax.json
\end{verbatim}

Наиболее важные файлы для включения в текст ВКР:
\begin{enumerate}
    \item \texttt{figure1\_replica.png} --- векторные поля и линии уровня энергии для ReLU и softmax;

    \item \texttt{flatness\_relu.png} и \texttt{flatness\_softmax.png} --- проверка плоских направлений энергии;

    \item \texttt{slr\_heatmap.png} --- визуализация значений SLR-критерия на сетке;

    \item \texttt{smith\_condition\_fraction.png} --- доля точек сетки, где выполнено условие Смита;

    \item \texttt{trajectories\_original\_relu.png}, \texttt{trajectories\_modified\_relu.png}, \texttt{trajectories\_original\_softmax.png}, \texttt{trajectories\_modified\_softmax.png} --- траектории исходной и модифицированной систем с отмеченными состояниями равновесия;

    \item \texttt{dynamics\_comparison.png} --- сравнение исходной и модифицированной динамики по сходимости;

    \item \texttt{equilibria\_relu.csv} и \texttt{equilibria\_softmax.csv} --- таблицы найденных состояний равновесия;

    \item \texttt{model\_parameters\_relu.json} и \texttt{model\_parameters\_softmax.json} --- сохранённые значения матриц \(W\) и векторов \(b\), использованных в конкретном запуске.
\end{enumerate}

Таким образом, разработанная программа позволяет воспроизводимо проводить вычислительные эксперименты, строить графики для отчёта и численно проверять основные эффекты, связанные с мёртвыми нейронами, плоскими областями энергии и модифицированной динамикой современной сети Хопфилда.
